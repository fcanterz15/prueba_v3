5.3 Redefinición de posiciones de los jugadores#

Como primer objetivo del Trabajo Fin de Máster, fue definido la redefinición de las posiciones de los jugadores de baloncesto a partir de las diferentes métricas estadísticas que estos obtuviesen a lo largo de una temporada. Para ello, a través de este notebook, será definido el proceso de refinición de posiciones y el análisis de estas nuevas posiciones. Para ello, será implmentado el algoritmo de clusterización k-meams con el que se espera obtener las nuevas posiciones a través de las estadísticas obtenidas. Además, previsamente a la implementación del algoritmo k-means, será realizada una reducción de dimensiones del conjunto de datos mediante el método PCA para la mejora del rendimiento del algoritmo de agrupamiento.

Teniendo esto presente, este notebook, estará dividido en los siguiente apartados:

  • 1. Reducción de dimensiones. A partir del algoritmo PCA, se realizará la reducción de dimensiones para mejorar el agrupamiento de clusterización.

  • 2. Obtención de agrupaciones. Se seleccionará el número obtimo de grupos en los que se puede englobar a los jugadores y se realizará la implementación del algoritmo k-means.

  • 3. Análisis de las nuevas posiciones. Se realizará un análisis de cada una de las agrupaciones con el fin de conocer como están distribuidos los jugadores y que perfil de jugadores se incluye en cada grupo.

A través de estos paso se espera por un lado obtener las nuevas posiciones y que se entienda todo el proceso implementado para su obtención.

1. Reducción de dimensiones.#

En este apartado, se procede a realizar la reducción de dimensiones que es una técnica empleada en análisis de datos y aprendizaje automático que permite disminuir el númer de variables en un cojunto de datos mientras se consera la mayor cantidad posible de información relevante. Como ha sido definido al principio de este notebook, ha sido dedicido realizar esta reducción de dimensiones mediante el algoritmo de Análisis de Componentes Principales o también conocido como PCA que es una ténica que permite la reducción de dimensiones mediante la realización de los siguientes pasos:

  1. Estandarización de variables. Se realiza la estandarización de los datos para que cada variable contribuya de manera equitativa al análisis mediante el uso de la media y dividiendo por la desviación estándar cada una de las variables.

  2. Cálculo de la matriz de covarianza. Se genera la matriz de covarianza en la que se mide cómo varian conjuntamente las diferentes variables del conjunto de datos.

  3. Cálculo de los componentes principales y sus valores. Se calculan los vectores de componentes principales y los valores que estos toman.

  4. Selección de los componentes principales. Se ordena cada uno de los componentes principales en función de los valores que estos tomas siendo los primeros componentes principales los que menor valor tienes.

  5. Transformación de los datos. Por último, se realiza la proyección de los datos originales sobre los nuevos ejes definidos por los nuevos componentes principales.

Mediante la implementación de estos pasos, son obtenidos los componentes principales que permiten la reducción dimensional de variables del conjuntos de datos principal.

2. Obtenición de agrupaciones#

Tras la reducción de dimensiones del conjunto de datos original, el siguiente paso a realizar es la obtención de agrupaciones en las que serán distribuidos los jugadores de baloncesto y que darán lugar a las nuevas posiciones de baloncesto. Para ello, este apartado estará dividido en dos secciones diferentes:

  • Selección del número óptimo de agrupaciones. Se realizará la selección del número óptimo de agrupaciones en las que será distribuidos los datos.

  • Aplicación del algoritmo kmeans. Se implementará el algoritmo k-means sobre el conjunto de datos reducido dimensionalmente para obtener las agrupaciones a las que pertenece cada uno de los jugadores.

2.1 Selección del número óptimo de agrupaciones.#

La selección del número óptimo de agrupaciones, o clusters, es la etapa dentro de la clusterización en la que se determina cual es el número de agrupaciones se ajusta mejor al conjunto de datos sobre el que se quiere aplicar el algortimo de clusterización. No existe ningún método que permita determinar el número de clusters debido a que la selección del número óptimo de clusters depende de la naturaleza de los datos y el objetivo del análisis. Es por este motivo, que para la obtención del número óptimo de clusters ha sido decidido que se utilice el Criterio de la silueta que es un método que permite media la calidad de los clusters a partir de los similares que son los puntos dentro de un cluster con respecto a otros puntos dentro de otro cluster. Es por este motivo, que ha sido realizado el cálculo de la puntuación de la silueta para diferentes número de clusters y la visualización de cada uno de esas siluetas:

# Se define número de agrupaciones a observar y se generan las visaulizaciones
range_n_clusters = [2, 3, 4, 5, 6, 7, 8]
for n_clusters in range_n_clusters:
    obtener_silueta(pca_8D_data, n_clusters)

Tras la extración de las siluetas para los valores de 2 a 8 clusters, se puede observar como la división del conjunto de datos en 5, 7 y 8 clusters no son correctas debido a su alto porcentjae de valores negativos además de que la distribución dentro de que existen siluetas para estos números de clusters que no superar el valor promedio de la silueta. Por otro lado, el número de agrupaciones de 2. 3 y 4 si que dan un porcentaje de error asumible los grupos que se generan son demasiado grandes y se aprecia que existen agrupaciones posibles dentro de esos cluster. Por lo tanto, el número de cluster óptimo que ha sido seleccionado es el de 6 debido a que el porcentaje de error es de 0.15 que es un valor asumible y todos los clusters tiene una distribución forma muy simétrica.

2.2 Aplicación del algoritmo kmeans#

Tras la obtención del número óptimo de clusters, se pasa a realizar la implementación del algoritmo de clusterización. Com ha sido definido anteriormente, se ha obtado por implemental el algoritmo k-means debido a su eficiencia computacional para el trabajo con conjuntos de datos y la facilidad a la hora implemtar que tiene el algoritmo. Con respecto al funcionamiento del algoritmo k-means este puede dividirse en los siguiente pasos:

  1. Inicialización. Selección de k puntos aleatorios del conjunto de datos como los centroides iniciales. El valor de k es dado al algoritmo debido a que es un algoritmo supervisado.

  2. Asignación. Asigna cada punto de datos al centroide más cercano.

  3. Actualización^. Recalcula los centroides de los k clusters utilizando la media de todos los puntos asignados al cluster.

  4. Repetición. Se repiten los paso 2 y 3 hasta que no se produzcan cambios significativos o se alcance el número maximo de iteraciones.

Tras definir el algoritmo se pasa a su inicialización y entrenamiento para 6 clústers y 8 componentes:

# Se inicializa el algoritmo k-means y se realiza su entrenamiento
algo_kmeans = KMeans(n_clusters=6, random_state=42)
algo_kmeans.fit(pca_8D_data)
