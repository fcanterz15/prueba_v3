5.2 Análisis exploratorio del Tablon estadísticas#

Como paso previo a la realización de los análisis y tras la construcción del tablón de estadísticas, se pasa a realizar el análisis exploratorio de este con el fin de conocer la estructura de los datos finales con los que trabajar y con se encuentran estos relacionados. Por lo tanto, en este notebook, se pasa dar respuesta a esto mediante los siguientes puntos:

  • 1. Análisis de consistencia de los datos. Se realizará el análisis de volumetrías referentes a los datos descargados con el fin de tener una imagen global de estos.

  • 2. Análisis univariante de los datos. Se realizará una análisis de cada una de las variables para conocer la información que esta almacena y la distribución de esta dentro de la variable.

  • 3. Análisis de correlaciones. Se realizará un análisis de correlaciones para poder evaluar si existen una alta relación entre los datos seleccionados.

A través de estos puntos se espere conocer por completo la informaición que se tiene para el cumplimiento de los objetivos del Trabajo Fin de Máster.

2.17 Conclusiones del análisis univariante#

Como se ha podido observar en los distintos gráficos de cada una de las variables, todas ellas se encuentran dentro del rango de los valores de 0 a 1 por lo que no se encuentran valores incoherentes dentro de los datos. Por otro lado, con respecto a la variabilidad de las variables, se ha observado como estas pese al filtrado de valores mantienen una variabilidad alta lo que permitirá poder distinguir a los jugadores en función de las estadísticas seleccionadas. Por último, se observa como en la mayoría de variables tiende a valores cercanos a cero salvo las variables relacionadas con el tiro.

3. Análisis de correlaciones#

Tras el análisis univariante de cada una de las variables que conforman el Tablón de estadísticas se pasa a realizar el estudio de correlaciones con el fin de ver como se relacionan las variables entre sí:

# Selección de variables numéricas con las que evaluar la correlacion entre estas
df_corr = df_tablon_estadisticas[['ORB%','DRB%', 'STL%', 'BLK%', 'TOV%', 'AST%','FTr','TL%', 'PINTURA_p', 'PINTURA_v', 'MID_RANGE_p', 'MID_RANGE_v','3FRONT_p', '3FRONT_v', '3CORNER_p', '3CORNER_v']]
corr = df_corr.corr()

# Se genera la visualizacion
sns.set(style="white")
plt.figure(figsize=(20, 8))
sns.heatmap(corr, mask=np.triu(np.ones_like(corr, dtype=bool)), fmt=".1f", cmap='coolwarm', linewidths=.5, vmin=-1, vmax=1, annot=True)
plt.title('Mapa de correlación de variables')
plt.show()

Como puede observarse, las correlaciones se encuentra estabilizadas y son asumibles. Sin embargo hay que destacar que la alta correlación entre la variable ORB% y las variables DRB%, BLK% y PINTURA_v se debe a que los jugadores interiores se caracterizan por la relación de estas tres variables por lo que es asumible.
